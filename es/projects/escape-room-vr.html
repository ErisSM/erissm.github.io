'Escape Room' en Realidad Virtual

Palabras clave: Unity, C#, VR

Descripción

Desarrollado como proyecto final de una asignatura del Máster en Computación Gráfica, Realidad Virtual y Simulación, en este proyecto se proponía la creación de una 'escape room' en realidad virtual, donde se tuviesen que ir superando pruebas hasta completar el juego.
Para el framework básico de realidad virtual volví a utilizar la API de OpenXR. En cuanto al juego en sí, planteé una escape room de ciencia ficción que contaba con 5 salas, cada una con su propio puzle que resolver. Al completar cada una de las actividades de estas salas se recompensaría con un disquete, que eran las llaves necesarias para completar el juego.
Entre los minijuegos a resolver se encontraban: encontrar un código secreto escondido entre objetos interactivos, una prueba de disparo con arco, una prueba de plataformas móviles controladas por palancas y un nivel con un jetpack que te permitía ascender verticalmente mediante impulsos.
Además de la programación de las distintas interacciones, también se desarrollaron shaders para crear efectos visuales en algunas áreas y se implementaron técnicas de optimización, como desactivar las salas inactivas hasta que el jugador se acercase a la puerta para acceder a la misma.

Rol en el proyecto

Fui la desarrolladora única del proyecto, ya que se trataba de una entrega individual para la universidad.

Herramientas y tecnologías usadas

  • Unity 2020.3
  • C#
  • OpenXR
  • Shadergraph
  • Meta Quest 2
