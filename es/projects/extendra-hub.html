Extendra Hub

Palabras clave: VR, PC, Multijugador, C++, Unreal Engine

Descripción

Extendra Hub es una solución de realidad virtual diseñada para satisfacer las necesidades de colaboración de empresas y trabajadores. La plataforma permite a los usuarios que trabajan en proyectos en equipo, hacer coworking, organizar reuniones y realizar conferencias virtuales, facilitando una comunicación y colaboración eficientes.
Se trata de una aplicación multijugador multiplataforma que permite hasta 30 usuarios simultáneos en un mismo mapa, disponible para VR -estando publicada en la tienda de Meta Quest- y para PC.
La aplicación contaba con múltiples mapas orientados al trabajo grupal y a la realización de eventos y reuniones, equipados con herramientas útiles para ello: pizarras, notas, pantallas virtuales que comparten desde un PC, ventanas de passthrough al mundo real, posibilidad de cargar media personalizada desde el backend web... Y ofreciendo para los escenarios tanto ambientaciones formales como otras más desenfadadas.

Rol en el proyecto

Formé parte de un equipo formado por otros programadores de Unreal Engine, varios artistas encargados de realizar el diseño y modelado de los escenarios y un programador web que se encargó del dearrollo del backend de la aplicación.
Colaboré con el resto de programadores para implementar las distintas interacciones y tecnologías deseadas. Durante el desarrollo usamos la metodología ágil Kanban, con un tablero donde apuntábamos las tareas por orden de prioridad y actualizabamos su estado en 'dailys' diarios.
Entre las funcionalidades más importantes de las que me encargué en este proyecto se encuentran la implementación de hand-tracking como alternativa a los mandos para interactuar con la aplicación, la adaptación de algunas interacciones para que también pudiesen realizarse desde PC, la creación de una herramienta que abría una ventana de passthrough o la conexión con el backend mediante websockets.

Herramientas y tecnologías utilizadas

  • Unreal Engine 5.1
  • C++
  • Meta Quest 2 y 3
  • AWS & Gamelift
  • GitHub
