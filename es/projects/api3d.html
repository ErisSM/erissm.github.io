Creación de un motor gráfico en C++ usando OpenGL

Palabras clave: Motor gráfico, C++, OpenGL, Programación de shaders

Descripción

Se trató de un desarrollo realizado a lo largo de una asignatura completa del Máster en Computación Gráfica, Realidad Virtual y Simulación, sirviendo también como la prueba de evaluación de la misma.
El objetivo del proyecto era crear un motor gráfico que permitiese crear y visualizar escenas 3D, usando para ello la librería gráfica OpenGL 4. Las características que debía incluir el motor eran la creación de objetos 3D que constasen de una o más mallas de polígonos, a las que se les podría asignar color, material, textura y mapas de normales; luces puntuales o direccionales, cámaras, emisores de partículas y esqueletos para modelos animados. Todo ello se programó en un sistema de clases en C++ que inicializaba y realizaba las llamadas correspondientes a la librería gráfica para renderizar los objetos de la escena. También se desarrollarion shaders de vértices y fragmentos en GLSL para realizar los cálculos de color e iluminación.
Como última entrega del proyecto, también se implementó la librería gráfica Vulkan, pero no globalmente para todos los elementos del motor, únicamente permitiendo el renderizado de mallas sin iluminar.

Rol en el proyecto

Fui la desarrolladora única del proyecto, ya que se trataba de una entrega individual para la universidad.

Herramientas y tecnologías usadas

  • C++
  • OpenGL 4
  • GLSL
  • Vulkan
